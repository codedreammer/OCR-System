# TinyWorld OCR

A fully offline, ultra-light OCR system optimized for high-quality printed text.

## Features
- Offline OCR
- Template matching
- <10 MB size
- Low-end device friendly

## How to Run
1. Install dependencies:
   pip install opencv-python numpy
2. Place image in input_images/
3. Run:
   python main.py

## Limitations
- Printed text only
- No handwriting
- Font-dependent accuracy
